# DigiSenda AI Web Portal

## Stack Tecnológico
- **Frontend Framework**: Next.js 15+ (App Router)
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS con paleta *AI Corporate Nexus*
- **Iconografía**: Lucide React
- **Animaciones**: GSAP, ScrollTrigger
- **Contenido y Blog**: MDX (`next-mdx-remote`)
- **SEO**: Soporte para Next.js Metadata, Structured Data JSON-LD
- **Deploy**: Vercel o similar (Recomendado).

## Descripción del Proyecto
DigiSenda AI es un portal web corporativo premium diseñado para ayudar a emprendedores hispanohablantes e internacionales a lanzar, escalar y gestionar sus empresas en los Estados Unidos. El portal ofrece:
- Creación de empresas LLC y obtención de EIN.
- Desarrollo Web y presencia visual online, incluyendo integraciones clave y marca.
- Generación y calificación de clientes (marketing orgánico y de pago) a través del ecosistema y plataforma interna *SynapLeads*.
- Gestión de finanzas, preparación de impuestos e inventario de transacciones.

## Estructura de Documentación
Consulta el directorio `/docs` para documentación técnica detallada:
- `/docs/architecture.md`: Flujo del landing, módulos de frontend y decisiones arquitectónicas.
- `/docs/design-system.md`: Definiciones de tokens de diseño, colores (`Deep Space`, `Neural Blue`, `AI Cyan`, `Silver`, `Graphite`), fuentes (Space Grotesk, Inter, Plex Mono) y reglas estéticas visuales.
- `/docs/animations.md`: Instrucciones y lógica en la biblioteca GSAP para un feeling cinematográfico.
- `/docs/branding.md`: Misión, tono de voz y pilar identitario de *DigiSenda AI*.
- `/docs/placeholders-checklist.md`: Lista técnica de los elementos provisorios usados durante la fase de desarrollo inicial para que el equipo reemplace antes de ambiente producitvo (assets visuales, enlaces stub, etc.)

## Iniciar el Proyecto
```bash
npm install
npm run dev
# Accede a http://localhost:3000
```
